PhxEcommerce is an e-commerce platform built using the Elixir/Phoenix framework. This project was created for educational purposes to help learn and explore the Phoenix framework and is not intended for production use.


## 🛠 Tech Stack

Language: Elixir (v1.15+)

Framework: Phoenix (v1.8.1)

Database: PostgreSQL (with Ecto)

Styling: Tailwind CSS


### **Installation**

Clone the repository:

```bash
git clone https://github.com/Null-logic-0/phx_ecommerce.git
```

```bash
cd phx_ecommerce
```

```bash
mix setup
```

**Start the Phoenix server:**

```bash
mix phx.server
```
